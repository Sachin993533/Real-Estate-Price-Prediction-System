🏡 Real Estate Price Prediction System

📌 Overview

This project is a Real Estate Price Prediction System that estimates property prices based on key features such as square footage, number of bedrooms, bathrooms, and location. It combines a simple predictive model with a clean and interactive web interface to provide instant price estimates.

✨ Features
🏠 Predict house prices based on user inputs
📍 Location-based price adjustment (Downtown, Suburban, Urban, Rural)
📊 Dynamic calculation using weighted formula
🎯 User-friendly UI with responsive design
⚡ Instant results without page reload
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Styling: Custom CSS with responsive layout
Logic: JavaScript-based prediction formula
Design: Background image + modern UI components
📂 Project Structure

⚙️ How It Works

The system calculates the predicted price using a formula based on:

Base price
Price per square foot
Price per bedroom
Price per bathroom
Location multiplier

👉 The prediction is computed dynamically in JavaScript using user inputs.
