# 🏡 Real Estate Platform & Price Predictor

A beautiful, responsive web application for finding, buying, and selling properties. This platform features a **Real Estate Price Prediction Tool** that provides instant property valuation estimates based on key features like square footage, location, and room counts.

## 🌟 Key Features

- **🏠 Property Listings:** Browse through various modern homes and apartments.
- **💰 Price Predictor Tool:** Instant price estimation using a dynamic JS-based algorithm.
- **🤝 Buy & Sell Options:** Dedicated sections for buyers and sellers without agency commissions.
- **📱 Fully Responsive Design:** Beautiful UI that works perfectly on desktops, tablets, and mobile devices.
- **⚡ Fast & Lightweight:** Built with pure HTML, CSS, and vanilla JavaScript (no heavy frameworks).

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3
- **Scripting:** Vanilla JavaScript
- **Icons & Typography:** Ionicons & Google Fonts (League Spartan)

## 📂 Project Structure

- `index.html` - The main landing page with hero sections and listings.
- `predict.html` - The standalone price prediction tool.
- `buy.html`, `sell.html`, `listing.html` - Pages for core real estate services.
- `about.html`, `contact.html` - Information and contact details.
- `style.css`, `predict.css` - Custom styling files.
- `script.js` - Interactive UI logic and DOM manipulation.

## ⚙️ How the Price Predictor Works

The system dynamically calculates the estimated property price based on user inputs without reloading the page. It utilizes a base price and adds value based on:
- Square Footage
- Number of Bedrooms & Bathrooms
- Location Multiplier (Downtown, Suburban, Urban, Rural)
